AES-256-GCM Decrypt Tool
-------------------
Client-side AES-256-GCM encryption and decryption tool built using the
Web Crypto API. All cryptographic operations are performed entirely
client-side â€” no data ever leaves your browser.

Features - Client-side AES-256-GCM authenticated encryption - Secure
random key and IV generation - Manual input of Key (HEX) and IV (HEX) -
Combined IV | Ciphertext format - Base64 ciphertext encoding - Clipboard
support - Automatic IV rotation - Mobile-friendly UI - Offline
functionality

Security Model Algorithm: AES-256-GCM 
Key Size: 256-bit 
IV Size: 96-bit
Authentication Tag: 128-bit 
Crypto Engine: Web Crypto API
